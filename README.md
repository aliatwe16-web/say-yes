# 💖 Mako Dinner Invitation Website

A fun, playful, and romantic single-page website inviting Mako to dinner ❤️

This project turns a simple question into an interactive mini love-game filled with animations, hearts, and a mischievous "No" button that refuses to behave 😄

---

## ✨ Features

- 💌 Romantic animated UI with soft pink, purple, and gold theme  
- 💖 Floating heart particles in the background  
- 😏 A "No" button that escapes when you try to click it  
- 😂 Random funny messages when the button runs away  
- ❤️ A satisfying "Yes" button with celebration effects  
- 🎉 Confetti, sparkles, and glowing animations on success  
- 📱 Fully responsive (works on mobile & desktop)  
- 🎶 Optional built-in music toggle (Web Audio API)

---

## 💡 How it works

- Clicking **"Yes ❤️"** triggers a full romantic celebration with animations and a success message:
  > "Yay! I can't wait to have dinner with you, Mako! ❤️"

- Trying to click **"No 😅"** makes the button run away, increasing its speed and chaos over time 😆

---

## 🛠️ Built With

- HTML5  
- CSS3 (animations + gradients)  
- Vanilla JavaScript (no frameworks)
